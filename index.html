Niji the Discord bot!

Niji has all of the moderation commands you will need and some more cool commands for fun, it will log all the things it does in a channel called audit-log if you want it to.

Features:

24/7 up time

Moderation commands:

Create & Delete roles

Create & Delete text and voice channels

Add & take roles from members

Mute & un-mute people

Warn people

Fun:

Random cat & dog pictures

Misc:

Get info about a server

Get info about Niji

Random number generation

Get the invite like for Niji

Suggest commands for Niji

Report issues with Niji
